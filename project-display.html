Project

Mobile App: Storm.lyy

Storm.lyy is a mobile weather app that starts by prompting users to log in or create a new account. Once logged in, the app requests location access to show local weather conditions on the static home page. Use the bottom left map button to explore interactive weather maps, and filter specific weather phenomena with the top right widgets. Swipe between screens to view detailed weather info for different cities using the top search bar. Access extra features like temperature unit toggles, accessibility options, and logout via the bottom right drawer menu.
